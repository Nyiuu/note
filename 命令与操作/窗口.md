### 默认按键绑定

  

| 按键               | 操作        |
| ---------------- | --------- |
| Meta +.          | 聚焦到下一个    |
| Meta +,          | 聚焦到上一个    |
| Meta + J         | 向下聚焦      |
| Meta + K         | 向上聚焦      |
| Meta + H         | 向左聚焦      |
| Meta + L         | 向右聚焦      |
| Meta + Shift + J | 向下 / 向后移动 |
| Meta + Shift + K | 向上 / 向前移动 |
| Meta + Shift + H | 向左移动      |
| Meta + Shift + L | 向右移动      |
| Meta + I         | 增大        |
| Meta + Dd        | 减小        |
| Meta + F         | 切换浮动状态    |
| Meta + \         | 切换到下一个布局  |
| Meta + \|        | 切换到上一个布局  |
| Meta + Return    | 设置为主窗口    |
| Meta + T         | 使用平铺布局    |
| Meta + M         | 使用单窗口布局   |
| 未绑定              | 使用伸展布局    |
| 未绑定              | 使用阶梯布局    |
